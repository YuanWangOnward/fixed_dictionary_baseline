This is a matlab code package for the Fixed-dictionary Baseline(FDB) MRI compressive sensing reconstruction algorithm. 
Written by Yuan Wang, New York University, 2016, yw1225@nyu.edu

Start with the demo.m and walk through it, you should be able to get an idea how it works.

%%%% IMPORTANT %%%%
At the beginning the demo.m, you need to set your Matlab working directory to the 'code' directory.
%%%%%%%%%%%%%%%%%%%
